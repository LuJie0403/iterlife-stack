# Reunion 产品概览

最后更新：2026-04-11

本文档浓缩 Reunion 当前的产品定位、核心能力和近期优先级。

## 1. 产品定位

- 产品名称：壹零贰肆老友记（IterLife）
- 形态：技术博客 + 个人知识库 + 轻量互动社区
- 核心目标：以低维护成本持续输出内容、沉淀知识、连接老友

## 2. 用户角色

- 超级管理员：通过 Git 提交文章并维护系统
- 普通访客：可直接阅读、搜索、浏览项目
- 登录用户：可评论、回复和参与互动

## 3. 核心能力

- GitOps 文章发布
- 文章阅读与分享
- 全站搜索
- 统一身份登录与基础互动
- 两级评论
- 开源项目展示

## 4. 当前产品基线

### 已形成稳定基线

- 内容系统已进入 `1.1.x`
- 文章发布、阅读、分享和个人主页能力已可正式交付
- 当前标准部署链路已经统一到 GHCR + webhook

### 当前仍需重点补强

- 文章同步链路稳定性与补偿能力
- 统一身份与权限体系
- 自动化测试与 CI 门禁
- 搜索体验增强
- Markdown 阅读体验增强

## 5. 近期优先级

### P0

- 统一身份模型与会话闭环
- Webhook 幂等与补偿
- 自动化测试基线
- 接口限流与防刷

### P1

- 搜索相关性与高亮
- 评论体验升级
- Markdown 渲染增强
- `/projects` 页面完善

### P2

- 监控与告警
- 备份恢复演练
- 发布回滚 Runbook 固化
